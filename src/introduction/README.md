# 前言 (Introduction)

欢迎来到《Rust 高频交易：从基础到实战》。

本书旨在填补市场上关于使用 Rust 构建生产级高频交易（HFT）系统的空白。我们将不仅仅停留在语言特性上，而是深入到底层原理——从 CPU 缓存一致性到网卡驱动，从无锁队列到 FPGA 交互。

这本书是我在准备HFT相关工作时学习的知识，覆盖了基础和领域编码实践，未来如果从事相关行业可能会继续补充实战技术。本文基础部分编写大部分由Gemini-3-Pro完成，我结合自己的实际情况，对自己不懂的或者认为需要补充的地方进行了补充。

## 本书路线图

本书分为七个部分，由浅入深地构建一个完整的 HFT 系统：

1.  **低延迟 Rust 基础 (Foundations)**: 重新审视 Rust，关注内存布局、Unsafe 黑魔法和 CPU 微架构。
2.  **核心基础设施 (Core Infrastructure)**: 构建高性能的无锁数据结构、日志和配置系统。
3.  **网络与 I/O (Network & I/O)**: 探索 io_uring, DPDK, AF_XDP 等内核旁路技术。
4.  **市场连接 (Market Connectivity)**: 解析 FIX, ITCH, SBE 等交易所协议，处理海量市场数据。
5.  **交易引擎架构 (Trading Engine)**: 设计风控、策略框架和执行算法。
6.  **优化与硬件 (Optimization & Hardware)**: 榨干硬件性能，进行微基准测试和性能分析。
7.  **回测与仿真 (Simulation)**: 构建事件驱动的回测引擎和历史数据回放系统。
